# Receive Admin Access

Secure administrative access to all client marketing platforms. Use the checklist below for quick tracking and the detailed inventory table to document credentials or access status.

**Security Note:** Do not store sensitive passwords directly in this document if it is shared publicly or insecurely. Use a password manager (e.g., 1Password, LastPass) or a secure shared document (e.g., OneDrive/SharePoint with restricted access) for actual credentials.

## Quick Checklist: Key Platforms

Ensure you have **Admin** or **Manager** level access to these core tools:

- [ ] Google Analytics 4 (GA4)
- [ ] Google Search Console
- [ ] Google Ads
- [ ] Google Tag Manager
- [ ] Google Business Profile
- [ ] Website CMS (WordPress, Webflow, etc.)
- [ ] Domain Registrar / DNS (or verify client IT handles this)
- [ ] Email Marketing Tool (Mailchimp, etc.)
- [ ] CRM (HubSpot, BuilderTrend, JobTread, etc.)
- [ ] Social Media Accounts (Meta Business Suite, LinkedIn, etc.)
