# Website Design Standards

## Checklist

- [ ]  Consistent application of brand standards
- [ ]  Ample white space throughout website
- [ ]  Properly formatted content throughout
- [ ]  Meets basic accessibility standards
- [ ]  Uses high-quality and high-resolution images and photography
- [ ]  Professional look/feel
