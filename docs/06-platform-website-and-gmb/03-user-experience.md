# User Experience Standards

**Social proof elements in place:**

**Notes**   Logos, testimonials, client photos, etc.

## Checklist

- [ ]  A modern and makes a positive first impression (look and feel)
- [ ]  Social proof elements in place
- [ ]  Intuitive navigation
- [ ]  Links and buttons are obvious and overt
- [ ]  Search functionality
- [ ]  Meets basic accessibility standards
- [ ]  Text is legible
