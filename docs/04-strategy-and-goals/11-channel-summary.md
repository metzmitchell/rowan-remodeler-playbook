# Channel Summary

| Overarching Goal(s) | Leading Indicator(s) | Lagging Indicator(s) |   |
| --- | --- | --- | --- |
| **New Customer Revenue** | First-Time Purchases | New Customer Revenue (Monthly vs. Annual) |  |
| **Repeat Customer Revenue** | Number of Campaigns, Number of Flows/Automations | Repeat Purchase Rate |  |
| **Channel** | Details & Objectives | Primary KPI & Target Range | Secondary KPIs & Target Range |
| **Website** | Describe what you need to do on your website to improve results. | i.e. conversion rate of 7% for all goals | Traffic greater than 10,000 unique visitors/month. |
| **Blog Content/Thought Leadership** | Describe the type, frequency, and purpose of your blog content. |  |  |
| **Email Marketing** | Describe what your goals will be, what automations you are/will set up, how often you’ll communicate to clients/customers.members and the public, how you’ll use email to promote new/unopened space, etc. |  |  |
| **Social Media** | Describe the content categories, frequency, purpose of each platform, goals, promotional strategy, and joint ventures/partnerships. |  |  |
| **Advertising** | Describe where you’ll be advertising, how much you’ll spend on each platform/service, the details of your targeting, goals, and more. |  |  |
| **Search Engine Optimization** | Outline your SEO strategy, including links to keyword focus, audit reports, competitive landscape, and any other relevant data your SEO person provides. |  |  |
| **Traditional/Offline Marketing** | Describe your print/offline/traditional marketing efforts, including who you want to reach and how your efforts will achieve your desired outcomes. |  |  |
| **Messaging and PR** | Describe your core messaging strategy, where you’ll focus your time and attention, and how it will lead to the result you want. |  |  |
| **Referral Marketing** | Describe how you’ll build a referral program into your business (if applicable). |  |  |
| **Third party directories/platforms** | Keep a list of where you’ll be featuring your business, what services you’re targeting, and what the expected results will be. |  |  |
