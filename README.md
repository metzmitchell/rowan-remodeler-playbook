# Rowan Remodeler Playbook (Docusaurus v3)

Docs-only Docusaurus site with autogenerated sidebar, light mode only, and Rowan branding (navbar title + favicon).

## Requirements
- Node 18+
- npm

## Install
```bash
npm install
```

## Local dev
```bash
npm start
```
- Opens http://localhost:3000
- Site is served at the root: http://localhost:3000/

## Build
```bash
npm run build
```
Outputs static assets to `build/`.

## Preview build locally
```bash
npm run serve
```
Serves the production build (defaults to http://localhost:3000).

## Deployment (Vercel)
- Build Command: `npm run build`
- Output Directory: `build`
- Site assumes `baseUrl = /`
- If hosting at `https://playbook.rowan.build/`, no subpath rewrite is neededâ€”just point the domain to this Vercel project.

## Notes
- Docs live at the site root via `routeBasePath: '/'`.
- Sidebar auto-generates from `docs/`.
- No blog or edit links are enabled.
- Navbar title uses the South Korea Serif font served from `static/fonts/`.
